下载工程
========

```dos
cd Desktop
git clone https://github.com/angular/angular-phonecat.git angular-phonecat
cd angular-phonecat
```


构建并启动工程
===============

```dos
npm start
```


运行e2e测试
============

```dos
npm run protractor
```

================
